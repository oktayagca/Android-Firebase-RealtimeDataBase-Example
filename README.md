# Mobiroller Task
This is an basic e-commerce application.

## Technologies and Libraries 
- Mvvm architecture
- Firebase Storage
- Firebase Real Time Data Base
- Dagger-Hilt
- Coroutines
- LiveData
- View Binding
- Navigation Component
- Glide 
- Lottie
- Ssp and Sdp
- View Pager 2, Recycler View
- LeakCanary
